### **Conclus√£o**

Os **gr√°ficos de linhas** s√£o fundamentais para visualizar e analisar dados ao longo do tempo, ajudando a identificar **tend√™ncias**, **sazonalidades**, **ciclos**, e **volatilidades**. ### **3. Gr√°fico de Linhas Empilhadas**

#### **Exemplo Pr√°tico: Composi√ß√£o de Vendas de Tr√™s Produtos ao Longo de um Ano**

```python

produto_b_cumsum = np.cumsum(produto_b)
produto_c_cumsum = np.cumsum(produto_c)

# Criando o gr√°fico de linha empilhada

plt.fill_between(meses, produto_c_cumsum, color='g', alpha=0.6, label='Produto C')
plt.fill_between(meses, produto_b_cumsum, produto_c_cumsum, color='r', alpha=0.6, label='Produto B')
plt.fill_between(meses, produto_a_cumsum, produto_b_cumsum, color='b', alpha=0.6, label='Produto A')

# Adicionando t√≠tulo e r√≥tulos
plt.title('Composi√ß√£o das Vendas de Produtos A, B e C - 2024')
plt.xlabel('Meses')
plt.ylabel('Vendas')
plt.grid(True)
plt.legend()

# Exibindo o gr√°fico
plt.show()
```
